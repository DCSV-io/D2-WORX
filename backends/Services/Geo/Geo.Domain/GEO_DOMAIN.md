# Geo.Domain

Domain model for geographic data including content-addressable locations, immutable contacts, WHOIS records, and reference data entities (countries, currencies, locales). Implements DDD aggregate roots with value objects.

## Files

| File Name                                                         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Location.cs](Entities/Location.cs)                               | Content-addressable location aggregate with SHA-256 hash ID, optional coordinates/address, and references to subdivision/country for automatic deduplication.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [Contact.cs](Entities/Contact.cs)                                 | Contact aggregate with ContextKey/RelatedEntityId for loose coupling, optional ContactMethods/Personal/Professional value objects, and LocationHashId FK.                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [WhoIs.cs](Entities/WhoIs.cs)                                     | IP geolocation aggregate with content-addressable hash from IP+year+month+fingerprint, ASN metadata, network flags (VPN/Tor/proxy), and temporal versioning.                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [Country.cs](Entities/Country.cs)                                 | Country aggregate with ISO 3166-1 codes, phone formatting, sovereignty relationships, and many-to-many with currencies/locales/geopolitical entities.                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [Subdivision.cs](Entities/Subdivision.cs)                         | Administrative division child entity with ISO 3166-2 code, short code extraction, and parent country FK.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [Currency.cs](Entities/Currency.cs)                               | Currency reference entity with ISO 4217 codes, display names, decimal places, and symbol for monetary formatting.                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [Language.cs](Entities/Language.cs)                               | Language reference entity with ISO 639-1 code, English display name, and native endonym.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [Locale.cs](Entities/Locale.cs)                                   | Locale entity combining language+country (BCP 47), used for date/time/number/currency formatting preferences.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [GeopoliticalEntity.cs](Entities/GeopoliticalEntity.cs)           | Supra-national organization entity (NATO, EU, USMCA) with type enum covering economic/political/military/cultural categories and member country many-to-many.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [BiologicalSex.cs](Enums/BiologicalSex.cs)                        | Enum for biological sex classification: Male, Female, Intersex, Unknown.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [GenerationalSuffix.cs](Enums/GenerationalSuffix.cs)              | Enum for name suffixes: Jr, Sr, I, II, III, IV, V, VI, VII, VIII, IX, X.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [GeopoliticalEntityType.cs](Enums/GeopoliticalEntityType.cs)      | Comprehensive enum with 25+ categories organized by type: General (Continent, SubContinent, GeopoliticalRegion), Economic (FreeTradeAgreement, CustomsUnion, CommonMarket, EconomicUnion, MonetaryUnion, BilateralInvestmentTreaty, DevelopmentAgreement, ResourceSharingAgreement), Political (PoliticalUnion, HumanRightsAgreement, EnvironmentalAgreement, GovernanceAndCooperationAgreement, PeaceTreaty, DemocracyPromotionAgreement), Military (MilitaryAlliance, ArmsControlAgreement, StatusOfForcesAgreement, PeacekeepingAgreement, SecurityCooperationAgreement, NonAggressionPact). |
| [NameTitle.cs](Enums/NameTitle.cs)                                | Enum for personal titles: Mr, Ms, Miss, Mrs, Mx, Dr, Prof, Sir, Lady, Lord, RtHon, Rev, Fr, Pr, Sr, Elder.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [GeoDomainException.cs](Exceptions/GeoDomainException.cs)         | Abstract base exception for all Geo domain exceptions, provides protected constructors with message and optional inner exception.                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [GeoValidationException.cs](Exceptions/GeoValidationException.cs) | Validation exception capturing ObjectTypeName, PropertyName, InvalidValue, and Reason for detailed domain validation error reporting.                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [Coordinates.cs](ValueObjects/Coordinates.cs)                     | Immutable lat/long value object with 5-decimal precision validation (~1.1m accuracy).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [StreetAddress.cs](ValueObjects/StreetAddress.cs)                 | Immutable 3-line street address value object with line length validation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [ContactMethods.cs](ValueObjects/ContactMethods.cs)               | Value object containing validated email and phone number collections with primary item convenience properties.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [EmailAddress.cs](ValueObjects/EmailAddress.cs)                   | Validated email value object with cleaned/lowercased value and user-defined label tags.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [PhoneNumber.cs](ValueObjects/PhoneNumber.cs)                     | Validated phone value object with digit-only cleaning and user-defined label tags.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [Personal.cs](ValueObjects/Personal.cs)                           | Value object for personal contact details (name, gender, DOB).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [Professional.cs](ValueObjects/Professional.cs)                   | Value object for professional contact details (title, company, department).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
